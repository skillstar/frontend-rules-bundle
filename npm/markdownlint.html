TIP

FRB 文档 规范

支持配套的 markdownlint 可共享配置。

安装 ​

需要先行安装 markdownlint：

bash
npm install frb-markdownlint-config markdownlint --save-dev

使用 ​

在 .markdownlint.json 中继承本包:

json
{
  "extends": "frb-markdownlint-config"
}
